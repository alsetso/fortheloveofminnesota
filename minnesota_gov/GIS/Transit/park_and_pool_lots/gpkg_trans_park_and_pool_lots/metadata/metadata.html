Section 1: Overview

Originator:Metropolitan Council

Title: Park and Pool Lots

Abstract: The Park and Pool Lots layer contains an inventory of park and pool lots throughout the Twin Cities Metropolitan area and surrounding counties, including communities in the neighboring state of Wisconsin. These locations allow commuters the ability to park their private vehicles for extended periods of time and meet others for commuter pooling purposes.

The dataset was geocoded using TLG centerline streets. When specific site details were available, the geography was moved off street lines to actual facility locations using orthophotographic imagery as a base layer. All attempts were made to correctly reflect the real lot locations; however the positional accuracy of this data cannot be guaranteed.

This inventory is maintained as a geographic record of the parking lots locations, and can be utilized alongside the "Park-and-Ride Annual Survey" table for historic capacity and utilization figures of park and pool locations.

Purpose: To identify park and pool locations in the Twin Cities metropolitan area and surrounding counties, including communities in the neighboring state of Wisconsin.

Section 5: Attributes

Overview: Park and Pool Lots Attributes

TCode: Unique value used by Metro Transit for identification of facilities

PnRNum: Identification number for park and pool used in the Annual Survey Table

Name: Name of the park and pool lot

Status: Current status of the facility
Open = Facility is open for use
Closed = Facility is permantly closed

Address: Parking lot address

City: City in which parking lot is located

Provider: State DOT that maintains park and pool lot

LocDetails: Lot location details (intersection, quadrant, portion of parking lot, etc.)

YearOpen: The year in which this lot opened for service

YearClose: The year in which this lot has closed down, if applicable

CurrentCpct: Number of parking spaces currently available at this lot

SharedUse: Indicates whether the lot is shared with another use
Y = Yes, lot is shared with another use
N = No, lot is standalone and not shared with another use

ShrUseDesc: Description of the shared or complementary use of the parking lot, if applicable

Notes: Any additional notes regarding this parking lot
